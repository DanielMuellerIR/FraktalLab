## Reicht der Dateiname mit Autor?

**Nein, ich würde sagen: nicht ganz.** Der Dateiname wie

``` overflow-visible!
BotB 14870 MovieMovies1 - what is this thing.sid
```

enthält vermutlich:

``` overflow-visible!
BotB [Entry-ID] [Autor] - [Titel].sid
```

Das ist gut, weil Autor und Titel sichtbar sind. Für Creative-Commons-Attribution ist aber normalerweise mehr sinnvoll: **Titel, Autor, Quelle und Lizenz**. Das wird oft als TASL-Prinzip beschrieben: Title, Author, Source, Licence. Eine Attribution sollte also nicht nur den Namen enthalten, sondern auch Quelle und Lizenz nennen.

Also besser nicht nur:

``` overflow-visible!
BotB 14870 MovieMovies1 - what is this thing.sid
```

sondern z.B.:

``` overflow-visible!
what is this thing.sid — MovieMovies1, Battle of the Bits Entry 14870, licensed via BotB as Creative Commons Attribution-NonCommercial-ShareAlike. No changes made.
```

Noch besser mit Links:

``` overflow-visible!
<a href="https://battleofthebits.com/arena/Entry/what+is+this+thing.sid/14870/">
  what is this thing.sid
</a>
— MovieMovies1,
from Battle of the Bits,
licensed as Creative Commons Attribution-NonCommercial-ShareAlike
(<a href="https://battleofthebits.com/lyceum/View/BotB%2BCC%2BLicense/">BotB license info</a>).
No changes made.
```

## Sollte „License see battleofthebits.com/license“ reichen?

Als Minimalvariante: **besser als nichts, aber ich würde es konkreter machen.**

Ich würde pro Song mindestens anzeigen:

``` overflow-visible!
Titel — Autor, from Battle of the Bits, used under Creative Commons Attribution-NonCommercial-ShareAlike according to BotB license information. Source: [Entry-Link]. License info: [BotB CC License]. No changes made.
```

Für deinen konkreten Beispielsong:

``` overflow-visible!
what is this thing.sid — MovieMovies1, from Battle of the Bits, Entry 14870, used under Creative Commons Attribution-NonCommercial-ShareAlike according to BotB license information. No changes made.
```

Plus Link auf:

``` overflow-visible!
https://battleofthebits.com/arena/Entry/what+is+this+thing.sid/14870/
```

und:

``` overflow-visible!
https://battleofthebits.com/lyceum/View/BotB%2BCC%2BLicense/
```
